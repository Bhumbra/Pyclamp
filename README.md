# Pyclamp

## INSTALLATION:

Install package python3-PyQt4 for your GNU/Linux distro then run:

```sh
$ pip3 install --user .
```

If your pip version is out of date, it needs to be upgraded (version>=20.0)
before installing

```sh
$ pip3 --version
$ pip3 install -U pip
$ pip3 --version
$ pip3 install --user .
```

## RUNNING:

```sh
$ ./run_pyclamp
```
