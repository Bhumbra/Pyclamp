3.7 Amplitude distributions

A window displaying amplitude distributions from each condition of release probability uses a common bin width that is shown in the top left. The BQA model profile for the amplitude distributions if overlayed in red. The model profiles can be exported using the button on the bottom right. If this button is clicked, Pyclamp exports a tab-delimited file with a variable number of columns. The first column contains the amplitude parameter whereas the remaining columns tabulate the estimated model profile for the selected width for each of the conditions of release probabilities. Note that the amplitude histograms themselves are not exported.
